## 1. 설치

### HA 애드온 (권장)

1. Home Assistant → 설정 → 애드온 → 애드온 스토어
2. 우측 상단 ⋮ → **저장소 추가** → `http://192.168.1.199:3100/yunjhl/choresync-ha-claude` 입력
3. **ChoreSync** 검색 후 설치
4. 설정 탭에서 옵션 입력 후 시작

```yaml
secret_key: "랜덤-32자-이상-문자열"
demo_seed: false
log_level: info
mqtt_broker: ""        # MQTT 브로커 IP (선택)
mqtt_user: ""
mqtt_pass: ""
mqtt_port: 1883
ha_tts_entity: ""      # 사용 안 함 (자동 탐지)
scheduler_enabled: true
```

5. **웹 UI 접속**: `http://[HA_IP]:8099`

> **주의**: 애드온 재설치 시 DB가 초기화됩니다. 설정 변경은 재시작으로 충분합니다.

## 2. 최초 설정

### 계정 생성

첫 실행 시 회원가입 화면에서 계정을 만듭니다.

## 3. 할일 관리

### 템플릿 생성

**할일** 탭 → **템플릿** 탭 → **+ 새 템플릿**

| 항목 | 설명 |
|------|------|
| 이름 | 할일 이름 |
| 카테고리 | 청소/요리/빨래 등 |
| 예상 시간 | 분 단위 |
| 강도 | Light/Normal/Hard (점수 배율에 영향) |
| 반복 주기 | 없음 / 매일 / 매주 / 매월 |
| 반복 요일/일 | 주간은 요일(0=월), 월간은 일자 |

> 마켓플레이스에서 임포트한 템플릿은 목록에서 자동으로 숨겨집니다.

### 점수 계산

```
점수 = round(예상시간(분) / 5 × 강도배율, 2)
```

| 강도 | 배율 |
|------|------|
| Light | 1.0 |
| Normal | 1.5 |
| Hard | 2.0 |

### 할일 완료

할일 카드 → **완료** 버튼 → 사진 첨부(선택) → 점수 자동 적립

### 반복 할일 자동 생성

`scheduler_enabled: true` 설정 시 매일 자정에 오늘 날짜에 해당하는 반복 할일이 자동 생성됩니다.

## 4. 퀘스트

특별 임무에 보상 포인트를 걸어 도전하는 기능입니다.

| 상태 | 설명 |
|------|------|
| Open | 수락 가능 |
| InProgress | 진행 중 |
| PendingReview | 완료 요청, 검토 대기 |
| Completed | 승인 완료, 포인트 지급 |
| Cancelled | 취소 |

**흐름**: 퀘스트 생성 → 수락 → 완료 요청 → 어드민 승인 → 포인트 지급

## 5. 위시리스트 (소원 투표)

가족이 함께 투표해서 소원을 승인/거절하는 기능입니다.

### 소원 등록

위시 탭 → **+ 소원 추가** → 제목, 설명, 포인트 비용 입력

### 투표

- 소원 카드의 **투표하기** 버튼 클릭
- 승인 / 거절 선택 (댓글 선택)
- 가족 과반수 승인 시 → `Approved` 상태로 자동 전환
- 가족 과반수 거절 시 → `Rejected` 상태로 자동 전환

### 상태 흐름

```
Pending → (과반수 승인) → Approved → (이행) → Fulfilled
        → (과반수 거절) → Rejected
        → (취소)        → Cancelled
```

## 6. IoT 연동

### MQTT 기기 등록

**IoT** 탭 → **+ 기기 추가**

| 항목 | 예시 |
|------|------|
| 이름 | 세탁기 |
| 타입 | Sensor |
| MQTT 토픽 | `choresync/washing_machine` |

### HA 자동화 연동

**IoT** 탭 → **HA 연동** 버튼

- **기존 템플릿 연결**: 드롭다운에서 기존 할일 템플릿 선택 후 HA 엔티티 연결
- **새 템플릿 생성**: HA 엔티티 기반으로 새 템플릿 자동 생성

생성된 자동화 YAML을 HA `automations.yaml`에 붙여넣으면 MQTT 신호 수신 시 자동으로 할일이 생성됩니다.

## 7. TTS 음성 알림

대시보드에서 가족에게 음성 메시지를 보낼 수 있습니다.

### 사용법

1. 대시보드 → **TTS 알림** 섹션
2. 발화할 텍스트 입력
3. 스피커 선택 (가족 중 누가 들을지)
4. TTS 엔진 선택 (아래 참조)
5. **발화** 버튼

### TTS 엔진

| 엔진 | 상태 | 비고 |
|------|------|------|
| 자동 (추천) | ✅ 동작 | google_translate_en_com 자동 선택 |
| Google Translate | ✅ 동작 | 한국어 지원, 약간 기계음 |
| Google AI TTS | ⚠️ 설정 필요 | Gemini API 키 필요 |

### Google AI TTS 활성화 방법

더 자연스러운 한국어 음성을 원할 경우:

1. [Google AI Studio](https://aistudio.google.com) → API 키 발급 (무료)
2. HA → 설정 → 통합 → **Google Generative AI** 추가
3. API 키 입력 후 저장
4. 이후 TTS 엔진 드롭다운에서 **Google AI TTS** 선택 시 동작

### 스피커 목록

`unavailable` 상태 및 중복 기기는 자동으로 필터링됩니다.

| 이름 | entity_id |
|------|-----------|
| 알파룸 | media_player.nestaudio4380 |
| 거실 | media_player.nestaudio8068 |
| 안방 | media_player.nesthubdb29_2 |
| 승민방 | media_player.googlehome5151 |
| 재하방 | media_player.googlehome0407 |
| 애플티비 | media_player.aepeultibi |

## 8. 마켓플레이스

커뮤니티가 공유하는 할일 템플릿 모음입니다.

- **마켓플레이스** 탭 → 원하는 템플릿 **임포트**
- 임포트된 템플릿은 **할일 목록에는 표시되지 않고** 내부 분류만 됩니다
- 필요 시 **할일 → 템플릿 탭**에서 직접 활성화 가능
